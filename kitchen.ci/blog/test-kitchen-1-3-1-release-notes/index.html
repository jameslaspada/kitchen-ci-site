Test Kitchen 1.3.1 Release Notes

by Fletcher Nichol on Friday, January 16 2015

For immediate release: Test Kitchen 1.3.1 is available on RubyGems.

Happy Friday! A quick release today to address an issue that was reported shortly after 1.3.0 went out the door. Given that was a straight up bug, it seemed like a good idea to remedy it as soon as possible.

In the meantime, the Readme was upgraded with some pretty SVG badgess.

Thank you to Mike Fiedler and Torben Knerr for your attention to detail, and accuracy!
